# simple-interest-calculator

This is a simple project designed to calculate simple interest based on the principal amount, annual interest rate, and time period.

## Formula Used
The formula used to calculate simple interest is:
A = P * (1 + R * T)

Where:
- **A**: Total Accumulated Amount (Principal + Interest)
- **P**: Principal Amount
- **R**: Annual Interest Rate (in decimal form, e.g., 5% = 0.05)
- **T**: Time Period (in years)

## Features
- Input principal amount.
- Input annual interest rate.
- Input time period in years.
- Instantly calculate and display the total interest and final balance.
